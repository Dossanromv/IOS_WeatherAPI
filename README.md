# iOS Weather App 🌤

A simple iOS weather app built with SwiftUI and REST API.

## Features
- Real-time weather data
- Temperature display
- Weather icons (☀️⛅️🌧️❄️⛈️)
- Dynamic background based on weather
- Switch between cities (Shymkent, Almaty, Astana)

## Tech Stack
- Swift 5
- SwiftUI
- REST API (Open-Meteo)
- async/await
- JSONDecoder

## What I Learned
- Making API requests with URLSession
- Parsing JSON with Codable
- async/await in Swift
- Dynamic UI updates with @State
